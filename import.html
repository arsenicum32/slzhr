Миссия

Фонд развития промышленности (ФРП) выделит ПАО "Химпром" 500 млн рублей на подготовку проектной документации для строительства производства гербицидов, сообщает пресс-служба "Группы Оргсинтез".

Экспертный совет ФРП одобрил заявку на выделение средств для разработки проектной документации для нового комплекса по производству глифосата, который предполагается построить на территории "Химпрома".
